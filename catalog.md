# Recipe Catalog

A map of every area where a system could exist, what's currently in place, and whether it works. This is the starting point for knowing what to fix and what to leave alone.

## Status key

| Status | Meaning |
|--------|---------|
| **Works** | Confirmed recipe — tested, consistent, earns its place |
| **Works when done** | Behavior works; the problem is consistency/anchoring |
| **Partial** | Something in place but gaps or optimization needed |
| **Gap** | No working system exists here |
| **Keeps failing** | Tried multiple times, hasn't stuck |
